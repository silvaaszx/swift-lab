## 🔬 Sobre o Laboratório

Este repositório é o meu ambiente de testes diário. O objetivo aqui não é construir aplicações completas, mas sim isolar problemas, testar novas APIs da Apple, aprimorar a lógica com Swift e construir componentes visuais complexos usando SwiftUI e UIKit.

## 📁 Estrutura

Cada pasta dentro deste repositório representa um estudo de caso independente:
* Nomenclatura das pastas: `[Nome-do-Estudo]` (Ex: `/Login-Screen-SwiftUI`, `/CoreData-Test`, `/API-Fetch-Combine`)
* Para rodar, basta clonar o repositório e abrir o ficheiro `.xcodeproj` correspondente à pasta do estudo.
